This is a demo for a simple election software package written in Python/Flask.

The demo is a complete application minus a few features so the demo application can run out of the box.  The demo application uses SQLite which I would never use in a production environment.  SQLite is included in the Python Standard Library.  sqlite3 — DB-API 2.0 interface for SQLite databases — Python 3.12.3 documentation 

This application consists of 4 Flask Blueprints in order to break the application up in manageable pieces.  There is currently only 1 static directory.

The application uses Python Logging and the demo application currently logs to the console only.
